user-api

MyroniX User Data API — handles profiles, watchlist, history, comments, chat, trivia, announcements, settings, and analytics.

Base path: /api/v2

  • /api/v2/user — Profiles, watchlist, watch history
  • /api/v2/comments — Anime & episode comments
  • /api/v2/chat — Global chat messages
  • /api/v2/trivia — Trivia scores & leaderboard
  • /api/v2/announcements — Site announcements
  • /api/v2/settings — Global app settings
  • /api/v2/analytics — Admin analytics
